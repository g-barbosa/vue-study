## Como rodar

``` bash
# instalar as dependências
npm install

# rodar o servidor em localhost:8080
npm run dev

# buildar
npm run build
```

For detailed explanation on how things work, consult the [docs for vue-loader](http://vuejs.github.io/vue-loader).
