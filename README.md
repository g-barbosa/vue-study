## Como rodar

``` bash
# instalar as dependências
npm install

# rodar o servidor em localhost:8080
npm run dev

# buildar
npm run build
```

Para uma explicação detalhada de como as coisas funcionam, consulte: [docs for vue-loader](http://vuejs.github.io/vue-loader).
